Q&A

Mention three main differences between Web1.0, Web2.0 and Web3.0:

Web1.0 Web2.0 Web3.0
Read-only static web. Read-write interactive web. Read-write intelligent web.
Professionally developed stand-alone applications. User-developed open applications. User-developed smart applications.
Syntax-aware basic browsing and search capabilities. Syntax-aware advanced browsing and search capabilities. Semantic-aware and context-aware next generation browsing and search capabilities.


Explain the differences using an example of a use case:
Say a user wants to look for information regarding side effects of drug X and is also interested in other drugs with the same side effects.
In web1.0 the user would have to go to a specific site that contains such information. The user must know the site in advance and its credibility.
However in web3.0, the internet is indexed and parsed constantly, making searching easier using semantics search, relevance and scoring.
Say the user searches on Google "What are the side effects of drug X", the user will get as a result many websites that are likely more credible thanks to advanced ranking algorithms.
Let's say as a part of the use case: that the user is also interested in other drugs that cause the same side effects as drug X. Because web3.0 sites allow various ways to collect data, some sites offer suggestions to other pages related to the pages users have visited in the past. Thanks to that, the same site that user has selected on his first search, shows at the end of the page a link that suggests the user to "View other drugs with the same side effects".
These kind of suggestions to users are based on various data collected from the site in and are possible thanks to Web3.0.
